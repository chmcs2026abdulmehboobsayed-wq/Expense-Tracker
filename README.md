# Personal Expense Tracker

## Project Description
Personal Expense Tracker is a web application that helps users track and manage their daily expenses.  
Users can add expenses with details like title, amount, category, payment method, and date.  
The system also provides monthly and category analysis to help users understand their spending habits.

## Features
- Add new expenses
- Edit and delete expenses
- View total expenses
- Monthly expense analysis
- Simple and user-friendly interface

## Technologies Used
Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Project Structure
Frontend:
- components – reusable UI components
- pages – main pages like HomePage and CreatePage
- lib – API configuration (Axios)

Backend:
- routes – API routes
- models – database schema
- controllers – business logic
